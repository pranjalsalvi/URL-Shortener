# URL Shortener 🚀

A modern and responsive **URL Shortener Web App** built using **Python & Flask** that allows users to convert long URLs into short, shareable links instantly.

---

## 🌟 Features

- Shorten long URLs with a single click
- Redirect users from short links to the original URLs
- Store URLs efficiently in a database
- Handle invalid URLs gracefully
- Responsive UI for desktop and mobile
- Clean and organized project structure

---

## 🛠 Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** SQLite / MySQL  
- **ORM:** SQLAlchemy  
- **Version Control:** Git & GitHub  

---
## 📂 Project Structure
```
URL-Shortener/
├── app.py               # Flask main app
├── models.py            # Database models
├── templates/           # HTML templates
├── static/              # CSS, JS, images
├── requirements.txt     # Project dependencies
├── .gitignore           # Git ignore file
└── README.md            # Project documentation
```
## Run the Flask app:
```
python app.py
```

## Usage

- Paste a long URL into the input field
- Click Shorten
- Copy the generated short URL and share
- Access the original URL via the short link
